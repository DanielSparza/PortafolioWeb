MI TRABAJO

BoxBook

Es una plataforma web creada en NodeJS para el registro de lecturas de libros donde los usuarios pueden actualizar sus lecturas diarias, pueden comentar y calificar acerca de un libro, y pueden compartir con otros usuarios de la plataforma.

Mantenimientos de equipo de computo

Es una plataforma web creada en NodeJS para el control de mantenimientos de equipos de computo la cual cuenta con dos tipos de usuario, el usuario común que es el que solicita los mantenimientos de los equipos a traves de un ticket y es el que da el seguimiento, y el administrador quien es el que autoriza o no los tickets, entre otras cosas.

Preguntas y Respuestas

Es una plataforma web creada en NodeJS donde los usuarios pueden consultar preguntas relacionadas al area de tecnologia y ver las respuestas que otros usuarios han realizado, tambien pueden crear una cuenta y publicar sus propias preguntas y respuestas, asi como calificar las respuestas de otros usuarios.

Tacos "Don Alfred"

Es una aplicación para un restaurante de comida desarrollada en dos partes, la primera fue desarrollada como aplicación de escritorio en c# de acceso solamente para el administrador y la segunda fue desarrollada como aplicación web utilizando el framework de laravel.

Abarrotes Sandra

Es una aplicación de escritorio desarrollada en c# para un negocio de abarrotes local, cuenta modulo de venta al publico, compras de productos a proveedores, registro de usuarios y proveedores, modulo de estadisticas mediante graficas y un modulo para administrar los turnos de trabajo de los empleados.

Página informativa de Minecraft

Es una página web informativa acerca de las novedades del videojuego Minecraft, cuenta un apartado de foro donde se permite el registro y login de los usuarios para que puedan hacer publicaciones en el. Fue desarrollada con el framework laravel.

Compilador

Es un compilador que permite controlar un brazo robótico con cuatro grados de libertad, desarrollado en c#. Cuenta con un analizador léxico, sintáctico, semántico y un traductor el cual traduce las instrucciones para mover el brazo robótico al lenguaje c y asi pueda ser interpretado por una placa controladora arduino.
